# Farcaster Snaps

Snaps are simple, nimble apps embedded in Farcaster casts. They render as cards in the
feed and respond to user input — buttons, sliders, text — without executing any code on
the client. A snap server returns JSON; the Farcaster client renders it.

> **Beta:** Snaps are still in beta and may change significantly over the next few weeks
> or months.
