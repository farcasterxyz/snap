# Agent Skills

This directory contains agent-facing skill files for structured workflows in this repository.

## Rules for skills

- Skills should not duplicate information that exists elsewhere in the docs. Instead they should include a concise summary and a
  link to the full context.

## Available skills

- [`create-farcaster-snap/SKILL.md`](https://github.com/farcasterxyz/snap/blob/main/agent-skills/create-farcaster-snap/SKILL.md) — Go from a natural-language description to a deployed snap. Uses the `template/` starter, validates against `@farcaster/snap`, and deploys to [host.neynar.app](https://host.neynar.app).
